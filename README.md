# Geppetto NWB model interpreter

This Geppetto module contains a model interpreter for [NWB](https://crcns.org/NWB/Overview), a format for cellular-based, neurophysiology data.
Through this module Geppetto has the ability to parse NWB files and display them.

For information about how this fits into [Geppetto](http://www.geppetto.org/) refer to the umbrella project [org.geppetto](https://github.com/openworm/org.geppetto) on GitHub.
